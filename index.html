Ethnicity data is preloaded for all 11 members (Eurogenes K13). The coloured bands visible in the Chromosomes tab show the whole-chromosome average — useful for seeing which population dominates each chromosome, but not pinpointed to the exact base-pair position of each shared segment.

For position-level ethnicity: Go to GEDmatch → Admixture (Heritage) → "Paint differences between 2 kits, 1 chromosome". Run this between two members who share a segment on a specific chromosome — the painter shows exactly which population colour appears at that location. Record your observations in Notes → Chromosome Painter observations.

To update or re-paste any member's data: Run "Admixture Proportions by Chromosome" on GEDmatch, paste the results in the member's box below, and send the copied block to the project administrator to embed permanently.
